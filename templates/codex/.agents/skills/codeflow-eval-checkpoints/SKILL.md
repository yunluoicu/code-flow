# CodeFlow Eval Checkpoints

## 检查点

1. 计划完成后检查 scope。
2. 实现中每个子任务完成后检查测试。
3. Review 前执行 lint / test / check。
4. 发布前执行 codeflow check。
5. 对比有 skill / 无 skill 输出质量。

## 输出

- Checkpoint
- Command
- Result
- Risk
- Next Step
