# CodeFlow Review

代码审查流程。对当前分支改动执行完整 Code Review，输出五维度报告。

**使用场景**：代码改动完成后、提交前（/codeflow-review）。有代码逻辑改动时必须执行。

**输出格式**：Must Fix / Should Fix / Test Gap / Risk / Evidence。

始终使用简体中文回复。

本 Skill 只提供流程、检查项和输出规范，不自动执行 Git 写操作。
