# CodeFlow Discovery

现有能力发现。在新需求开始阶段搜索项目中已有的实现、API 接口、可复用组件、工具函数、现有模式，避免重复实现。

**使用场景**：新需求开始时的 discovery 阶段（简单需求轻量检查、复杂需求完整检查）。

**检查来源**：graphify-out/ 知识图谱、现有代码结构、已有 API/组件/工具函数。

始终使用简体中文回复。

本 Skill 只提供流程、检查项和输出规范，不自动执行 Git 写操作。
