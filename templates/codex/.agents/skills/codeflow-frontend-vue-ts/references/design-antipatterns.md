# Design Anti-patterns

禁止输出以下常见 AI UI 反模式：

1. 全页面都是圆角卡片，没有层级。
2. card 套 card。
3. 大面积渐变但没有信息结构。
4. 图标随机堆叠。
5. 表格没有筛选、空态、错误态。
6. 状态颜色不统一。
7. 所有文字同一字号。
8. 间距靠随机 margin 拼凑。
9. 表单错误提示不清楚。
10. 没有真实业务字段，只有泛化占位内容。

## CodeFlow UI 标准

- 专业、克制、工程感。
- 高级灰背景 + 清晰状态色。
- 信息密度适中。
- 状态和风险必须视觉可辨识。
- Dashboard 页面必须有 KPI、筛选、数据表、详情/上下文区。
