# CodeFlow Skill Learning

## 原则

1. 不自动写入 learnings。
2. 发现重复问题时提醒用户是否沉淀。
3. 写入前要求用户确认。
4. learning 必须包含场景、错误模式、正确做法、示例。
5. profile / index / sync 可读取 learnings。

## 推荐文件

- `.codeflow/learnings/frontend.md`
- `.codeflow/learnings/backend-go.md`
- `.codeflow/learnings/project-patterns.md`
- `.codeflow/learnings/review-findings.md`
