# CodeFlow Frontend Vue + TypeScript Skill

## 触发条件

当任务涉及以下内容时必须使用本技能：

- `package.json`
- `vite.config.ts`
- `nuxt.config.ts`
- `src/**/*.vue`
- `src/**/*.ts`
- `src/**/*.tsx`
- `components/`
- `pages/`
- `layouts/`
- `stores/`
- `router/`
- `tailwind.config.*`
- `uno.config.*`
- Dashboard / Admin / Web UI / H5 / 前端页面 / 组件开发

## 核心目标

输出专业、可维护、非临时 Demo 的前端代码。

要求：

1. 页面结构清晰。
2. 组件边界明确。
3. 状态管理可控。
4. 类型定义完整。
5. UI 视觉统一。
6. 交互状态完整。
7. 空状态、加载态、错误态齐全。
8. 不写一次性堆砌页面。
9. 不写“AI 味”明显的 UI。

开始前先检查：

## 输出要求

前端实现完成后必须说明：

- 修改的页面 / 组件 / composable
- API contract 变化
- loading / empty / error 状态处理
- 响应式处理
- 测试或手动验证方式
- UI 风险和后续建议
